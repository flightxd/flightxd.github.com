Property Detail
behaviorsproperty
behaviors:CompositeBehavior

A dynamic object or hash map of behavior objects. behaviors is effectively read-only, but setting either an IBehavior or array of IBehavior to this property will add those behaviors to the behaviors object/map. To set behaviors in MXML: <Component...> <behaviors> <SelectBehavior/> <ButtonBehavior/> </behaviors> </Component>

This property can be used as the source for data binding.
