Classpublic class Skin
InheritanceSkin flash.events.EventDispatcher
Implements ISkin, IContainer, IStateful, IMeasurable

Skin is a convenient base class for many skins, a swappable graphical definition. Skins decorate a target Sprite by drawing on its surface, adding children to the Sprite, or both.

Property Detail
contentproperty
content:IList

Holds the children to be measured and positioned by the container. Use children.addItem, children.removeItem, etc to add/remove items participating in measurement and layout. Note that you can still manipulate the DisplayList directly using the addChild, removeChildAt, etc. methods of DisplayObjectContainer, however children added manually will not be measured or positioned by the container.

This property can be used as the source for data binding.
